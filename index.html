How to read it. Each seam shows an inlier count — the number of matched rock features that agreed on the alignment. ≥25 is a firm lock, 10–25 usable, under that the seam fell back to a straight vertical stack and wants a look. Use the ± nudges to slide a frame and re-render. Registration allows translation, rotation and scale, so it corrects handheld tilt and distance drift — but it stays a 2-D mosaic for logging, not a metric orthophoto. For scaled 3-D, use SfM.
